# Clinic - Spring Boot
##  Description
* **Final project of the backend subject at [Certified Tech Developer](https://www.digitalhouse.com/ar/productos/programacion/certified-tech-developer).**


* **API REST standalone project.**
## What i use

* **Maven**
* **Spring Web** 
* **Spring Security**
* **Spring Data JPA**
* **JWT authentication**
* **Hibernate**
* **H2 Database**
* **Log4j**
* **JUnit**
* **Jackson**
* **HQL**
* **Lombok**
* **Bootstrap 5**
